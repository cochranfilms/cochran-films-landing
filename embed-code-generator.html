How to Use

  1. Choose an embed option above that fits your needs
  2. Copy the code using the "Copy Code" button
  3. Paste the code into your website's HTML where you want the Service Package Builder to appear
  4. Customize the dimensions by changing the width and height attributes
  5. Test the integration to ensure it works properly on your site

Customization Options

  • Width: Change width="100%" to any pixel value or percentage
  • Height: Adjust height="600" to fit your content area
  • Responsive: Use the responsive embed option for mobile-friendly layouts
  • Event Handling: Use the event handling option to capture quote submissions

Supported Browsers

Works in all modern browsers including Chrome, Firefox, Safari, and Edge.
